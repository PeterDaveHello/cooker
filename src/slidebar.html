Usage

Markup

{% highlight html %} // Position Right
Slidebar Right

Slidebar Right

// Position Bottom
Slidebar Bottom

Slidebar Bottom

{% endhighlight %}

data-width,data-heightで開いた時のサイズを指定。

JavaScript

{% highlight js %} $('.slidebar').slidebar({ position: 'left', speed: 500, easing: 'easeOutExpo', minSize: 10, parentClass: '.tab-content' }); {% endhighlight %}

Methods

$().slidebar(options)

$().slidebar('open')

{% highlight js %}$('#element').slidebar('open'){% endhighlight %}

$().slidebar('close')

{% highlight js %}$('#element').slidebar('close'){% endhighlight %}

{% highlight js %}$('#element').slidebar('toggle'){% endhighlight %}

$().slidebar('destroy')

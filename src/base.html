Install {{ site.title }}

There must be installed git,node,npm,ruby,gem,jekyll,grunt,bower to the local machine.

{% highlight html %} $ cd my-site $ git clone https://github.com/blivesta/ashiba.git $ cd ashiba $ npm install $ grunt build {% endhighlight %}

Grunt Tasks

{% highlight html %} $ grunt build {% endhighlight %}

{% highlight html %} $ grunt　 {% endhighlight %}

Deploying to gh-pages branch the contents of
the master branch _gh_page directory.

Slidebar slidebar.js

イージング効果はjquery.easing.jsを利用。

Examples

Slidebar Right

Slidebar Right

Slidebar Bottom

Slidebar Bottom

Usage

Markup

// Position Right
<div class="slidebar slidebar-right" data-width="240">
  <div class="slidebar-toggle">Slidebar Right</div>
  <div class="slidebar-body" style="width: 0;">
    <div class="slidebar-resize"></div>
      <div class="slidebar-body-inner">
        <h3>Slidebar Right</h3>
      </div>
    </div>
</div>

// Position Bottom
<div class="slidebar slidebar-bottom" data-height="300">
  <div class="slidebar-toggle">Slidebar Bottom</div>
  <div class="slidebar-body" style="height: 0;">
    <div class="slidebar-resize"></div>
    <div class="slidebar-body-inner">
      <h3>Slidebar Bottom</h3>
    </div>
  </div>
</div>

data-width,data-heightで開いた時のサイズを指定。

JavaScript

$('.slidebar').slidebar({
  position: 'left',
  speed: 500,
  easing: 'easeOutExpo',
  minSize: 10,
  parentClass: '.tab-content'
});

Methods

$().slidebar(options)

$().slidebar('open')

$('#element').slidebar('open')

$().slidebar('close')

$('#element').slidebar('close')

$().slidebar('toggle')

$('#element').slidebar('toggle')

$().slidebar('destroy')

$('#element').slidebar('destroy')

      <td>...</td>
    </tr>
    <tr class="drilldown-body" data-level="4" style="display:none;">
      <td>...</td>
    </tr>

      <td>...</td>
    </tr>
    <tr class="drilldown-body drilldown-toggle" data-level="2" style="display:none;">
      <td>...</td>
    </tr>
    <tr class="drilldown-body" data-level="3" style="display:none;">
      <td>...</td>
    </tr>
  </tbody>

Markup

<div class="switchers" data-mode="accordion">

  </div>
  <div class="switcher">

</div>

$('.switchers').switchers({

  toggle:     'switcher-toggle',
  toggleAll:  'switcher-toggle-all',
  open:       'switcher-open',
  close:      'switcher-close'
});

Scrollmethod scrollmethod.js

クリックで要素までスクロールさせるJQueryプラグイン。jquery.easing.jsが必要。

Usage

Markup

<body>
  <a href="#target">Drower toggle</a>
  <div id="target"></div>
</body>

JavaScript

$('a[href^=#]').scrollmethod({
    speed:     800,
    easing:   'easeOutExpo',
});

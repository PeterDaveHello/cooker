Drawer drawer.js

Phone

Tablet

Desktop

Usage

Markup

<body>
  <button class="drawer-toggle">Drower toggle</button>
  <div class="drawer drawer-default">
    <nav class="drawer-nav" role="navigation">
      <div class="drawer-brand"><a href="#">Cooker</a></div>
      <ul class="nav drawer-nav-list">
        <li><a href="#">Nav list</a></li>
      </ul>
    </nav>
  </div><!-- /.drawer -->
  <div class="drawer-overlay">
    <h1>Page contents</h1>
  </div><!-- /.drawer-overlay -->
</body>

JavaScript

$('.drawer').drawer({
  nav:          'drawer-nav',
  navList:      'drawer-nav-list',
  overlay:      'drawer-overlay',
  toggle:       'drawer-toggle',
  openClass:    'drawer-open',
  closeClass:   'drawer-close',
  desktopEvent: 'mouseover', // or click 
  speed:        200,
  width:        280,
  bottomMargin: 80
});

Methods

$().drawer(options)

$().drawer('open')

$('#element').drawer('open')

$().drawer('close')

$('#element').drawer('close')

$().drawer('toggle')

$('#element').drawer('toggle')

$().drawer('resize')

$('#element').drawer('resize')

$().drawer('destroy')

$('#element').drawer('destroy')

Slidebar slidebar.js

イージング効果はjquery.easing.jsを利用。

Examples

Slidebar Right

Slidebar Right

Slidebar Bottom

Slidebar Bottom

Usage

Markup

// Position Right
<div class="slidebar slidebar-right" data-width="240">
  <div class="slidebar-toggle">Slidebar Right</div>
  <div class="slidebar-body" style="width: 0;">
    <div class="slidebar-resize"></div>
      <div class="slidebar-body-inner">
        <h3>Slidebar Right</h3>
      </div>
    </div>
</div>

// Position Bottom
<div class="slidebar slidebar-bottom" data-height="300">
  <div class="slidebar-toggle">Slidebar Bottom</div>
  <div class="slidebar-body" style="height: 0;">
    <div class="slidebar-resize"></div>
    <div class="slidebar-body-inner">
      <h3>Slidebar Bottom</h3>
    </div>
  </div>
</div>

data-width,data-heightで開いた時のサイズを指定。

JavaScript

$('.slidebar').slidebar({
  position: 'left',
  speed: 500,
  easing: 'easeOutExpo',
  minSize: 10,
  parentClass: '.tab-content'
});

Methods

$().slidebar(options)

$().slidebar('open')

$('#element').slidebar('open')

$().slidebar('close')

$('#element').slidebar('close')

$().slidebar('toggle')

$('#element').slidebar('toggle')

$().slidebar('destroy')

$('#element').slidebar('destroy')

Scrollmethod scrollmethod.js

クリックで要素までスクロールさせるJQueryプラグイン。jquery.easing.jsが必要。

Usage

Markup

<body>
  <a href="#target">Drower toggle</a>
  <div id="target"></div>
</body>

JavaScript

$('a[href^=#]').scrollmethod({
    speed:     800,
    easing:   'easeOutExpo',
});

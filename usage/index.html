Usage

Download

Zip File Download cooker Currently v0.1.4.

Install

If you want to compile, there needs to be installed on the local git, node, npm, ruby, gem, jekyll, grunt, a bower.

$ git version
$ node -v
$ npm -v
$ ruby -v
$ gem -v
$ jekyll -v
$ grunt -version
$ bower -v

Step 1

作業用ディレクトリで$ git clone。

$ cd working-dir
$ git clone https://github.com/teq-inc/cooker.git

Step 2

npmでgruntのモジュール（node_modulesフォルダ）をワーキングディレクトリにインストール。

$ cd cooker
$ npm install

Step 3

goコマンドでgruntとbowerで関連ファイルのインストール→ビルド→サーバー起動。

$ grunt go

Gruntfile.js

gruntタスクの実行コマンド。

Default

サーバー起動（+livereload） → ファイル監視（jekyll,less）

$ grunt

Build

ファイルの最適化 → ビルド

$ grunt build

Test

grunt build → エラーチェック

$ grunt test

Go

vendorファイルをインストール → grunt test → grunt build → grunt default

$ grunt go

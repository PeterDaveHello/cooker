Base layout

Grunt and jekyll that will be used as a scaffold of application development.

Install cooker

There must be installed git,node,npm,ruby,gem,jekyll,grunt,bower to the local machine. 

$ cd my-site
$ git clone https://github.com/blivesta/ashiba.git
$ cd ashiba
$ npm install
$ grunt build

Grunt Tasks

install vendor files → Build app

$ grunt build

Starting the local server → watching of file update → Livereload

$ grunt　

Deploying to gh-pages branch the contents of
the master branch _gh_page directory. 

$ grunt deploy

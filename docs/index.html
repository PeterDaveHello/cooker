install

ローカル環境に'git','node','npm','ruby','gem'をインストールされているか確認。

{% highlight text %} $ git -v $ node -v $ npm -v $ ruby -v $ gem -v {% endhighlight %}

確認できたら'jekyll','grunt','bower'をインストール

{% highlight text %} $ gem install jekyll $ npm install -g grunt-cli $ npm install -g bower {% endhighlight %}

Step 1

作業用ディレクトリでgit cloneかダウンロードしてファイルの準備。

{% highlight text %} $ cd my-site $ git clone https://github.com/blivesta/cooker-scaffolding.git {% endhighlight %}

Step 2

npmでgruntのモジュールを作業用ディレクトリにインストール。

{% highlight text %} $ cd cooker-scaffolding/ $ npm install {% endhighlight %}

node_modulesフォルダが作られる。

Step 3

gruntとbowerで関連ファイルのインストール→ビルドを行う。

{% highlight text %} $ grunt go {% endhighlight %}

インストールが成功するとブラウザが起動する。
